# Directional Stock Prediction: Tesla (TSLA) Next-Day Movement

## Problem Definition:

### Ziel
Das Ziel dieses Projekts ist die Vorhersage der Kursrichtung der Tesla-Aktie (TSLA) für den nächsten Handelstag.
Es handelt sich um ein Binary-Classification-Problem, bei dem das Modell lernt, basierend auf historischen Kursdaten und technischen Indikatoren vorherzusagen, ob der Schlusskurs am folgenden Tag steigen (1) oder fallen (0) wird.
